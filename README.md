# Week 4 — Data Visualization & Dashboarding

## Project: Superstore Sales Performance Dashboard

### Overview
An interactive sales performance dashboard built in Microsoft Power BI
using the Kaggle Superstore Sales Dataset (9,994 rows | 21 columns | 2014–2017).

### Dashboard Theme
Deep Navy & Gold luxury design with rounded card visuals and interactive slicers.

### KPIs Tracked
- Total Sales: $2.30M
- Total Profit: $286.40K
- Total Orders: 9,994
- Profit Margin: 12.47%

### Key Insights
- West region leads all regions with ~$500K in sales
- Technology is the most profitable category at 36.4% of total sales
- Sales peak sharply every November–December
- Canon imageCLASS is the single best-selling product
- Furniture has the lowest profit margin despite decent sales volume
- Consumer segment drives the majority of orders

### Tools Used
- Microsoft Power BI (Dashboard)
- Microsoft PowerPoint (Presentation)
- Dataset: [Kaggle Superstore Sales Dataset](https://www.kaggle.com/datasets/vivek468/superstore-dataset-final)
